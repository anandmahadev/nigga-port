# Contributing Guidelines

We welcome contributions to FlipaClip.SYS! To collaborate:

1.  **Fork** the repository and clone locally.
2.  Create a feature branch mapping your specific changes (`git checkout -b feature/cool-customization`).
3.  Implement clean neobrutalist structures following our unified styling systems.
4.  Ensure all browser tests run perfectly using browser developer consoles (`PORTFOLIO_DIAGNOSTICS.runAll()`).
5.  Submit a descriptive Pull Request (PR) for review.
